## Installation 🛠️

1. Clone the repository: `git clone https://github.com/jiptwo553900/RPS-bot.git`
2. Rename `.env.example`: `mv .env.example .env`
3. In `.env` edit `BOT_TOKEN=<put your Telegram bot token here>`
4. Start `bot.py`
